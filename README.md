# Weather App

## Description
This is a simple Python Weather App that fetches live weather information using the OpenWeatherMap API.

## Features
- Search weather by city name
- Displays temperature
- Shows weather condition
- Displays humidity
- Shows wind speed

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## Requirements
- Python 3.x
- requests library

## How to Run
1. Install the requests library:
   pip install requests

2. Open the weather.py file.

3. Replace:
   API_KEY = "YOUR_API_KEY"
   with your OpenWeatherMap API key.

4. Run the program:
   python weather.py
